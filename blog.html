What is a callback function and why will you use it?

When a function is passed as an argument of another function then, it is called callback function. To ensure that a function is only going to be executed after a specified action is completed and not before, we need callback functions. In addition, callback functions are required to avoid problems caused by missing data that are required by other methods.
